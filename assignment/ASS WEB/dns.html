Domain Name System (DNS)

The Domain Name System (DNS) is the phonebook of the Internet. Humans access information online through domain names, like google.com. Web browsers interact through Internet Protocol (IP) addresses. DNS translates domain names to IP addresses so browsers can load Internet resources.

How DNS Works
  • Request: You type a domain name into your browser.
  • Translation: The DNS resolver finds the specific IP address associated with that name.
  • Connection: Your computer connects to the web server at that IP address to load the site.
Key Components
  • DNS Resolver: The server that receives the initial query from your computer.
  • IP Address: A unique string of numbers that identifies each computer using the Internet Protocol to communicate over a network.
