HyperText Markup Language (HTML)

HTML is the standard markup language used to create the structure of web pages. It uses "tags" to annotate text, images, and other content for display in a web browser.

Technical Role
  • Structure: Defines headings, paragraphs, links, and lists.
  • Integration: Works alongside CSS for styling and JavaScript for functionality.
  • Semantic HTML: Uses tags that describe the meaning of the content (e.g., <article>, <footer>).
